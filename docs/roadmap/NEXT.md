# FlowHub Next Priorities

FlowHub v1.3 Beta is the official development baseline. Future work follows
this high-level priority order:

1. Seller productivity
2. Workspace usability
3. FlowHub Sheet
4. Marketplace integrations
5. Automation
6. Reporting
7. AI assistance

Detailed implementation planning belongs in individual feature design work and
is intentionally not maintained in this roadmap.
