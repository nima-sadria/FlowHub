# FlowHub Roadmap

For the public roadmap, see the repository-level [ROADMAP.md](../ROADMAP.md).

This document mirrors the current release phase:

- First public deployment polish
- One-command installer
- Canonical `/opt/FlowHub` deployment path
- Setup Wizard without connector configuration
- Connector configuration in Settings
- Integration Platform
- Unified Logging Platform
- Read-only safety model

Future work is planned only after Owner approval and audit.
