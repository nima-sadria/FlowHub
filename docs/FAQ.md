# FAQ

## Where is FlowHub installed?

`/opt/FlowHub`.

## Where do I configure WooCommerce and Nextcloud?

Settings.

## Does setup configure connectors?

No. Setup contains Welcome, Server Profile, Database, Admin Account, and Finish.

## Are writes enabled?

No. Apply, scheduler execution, automatic pricing, WooCommerce writes, and
spreadsheet writes are disabled in the first release.

## What is `/opt/flowhub`?

Legacy Compatibility path for older installations. New installations use
`/opt/FlowHub`.
